falando sobre tag

inicio

o html tem multiplas tags que ajudam a compor sua estrutura, como a tag (i) que transformar o texto selecionado em italico, assim como a tag (strong), que por sua vez transformar o texto em negrito, lembrando que toda vez que essas tags forem abertas elas tem que conter no fim sua tag de fechamento.
No entanto tags como a tag imput, ela não tem tag de fechamentos pois quando usada ela cria um campo digitavel para o usuario, tambem tem a tag img que adiciona um imagem tambem não tem tags de fechamentos, entre outras tags que tambem não possuem.

Atributos das Tags

inicio

nos atributos das tags, temos alguns exemplos, como o atributo id, ele funconar como um identificador unico no seu texto, o atributo style, por sua vez, tem como objetivo atribuir estilos ao seu texto usando CSS, o atributo type, usado para comprementar o tag input mais usadas em formularios, o atributo src, é usando tambem como complemento da tag img, sua função é localizar a imagem que sera inserida.

textos

inicio

nesse paragrafos aborto algumas das tipologias de tags de textos, começando pelo H ele tem seis niveis de tamamhos e e usado para titulos desde o maior h1 ate o h6 que é seu menor nivel de tamanho. o p define os paragrafos do textos, muito usado no html, o blockquote é usado para citações, a mark é usada para destaque em amarelo, entres outras dezenas de tags que podem se usadas para criarseus texto html.

Listas Ordenadas e Não Ordenadas

inicio

no html poder ser usadas dois tipos de listas, as ordenadas e não ordenadas. As ordenadas sao usada as tags ol e dentro dessa ol fica outra tag a li que va definir as itens que compõem essa lista, ela pode ser ordedas em sequencia numerica, alfabetica, ja para criar uma lista nao ordenadas os passos sao os mesmos, so que ao inves de usar a tag ol vai ser usada a tag ul, essa lista nao tem sequencia definida.

inicio
